O mnie

Witaj. Nazywam się Marcin Galanty i od kilku miesięcy zgłębiam tajniki front-end web developmentu. Zaczynałem swoją przygodę od poznawania podstaw składni html i css. Następnie uczyłem się bardziej zaawansowanych znaczników css. Obecnie pracuję nad udoskonalaniem swojego warsztatu w obsłudze GITa oraz uczę się pracować nad projektem w Scrumie przy pomocy aplikacji jira. Wszystko to odbywa się w ramach bootcampu prowadzonego przez lubelski oddział infoShareAcademy. Masę wiedzy i praktyki otrzymywaną podczas codziennych 8-godzinnych warsztatów trzeba jednak szlifować jeszcze po godzinach w domu. W codziennej systematycznej pracy z zaangażowaniem rzeczy, które wydawały mi się nie do ogarnięcia stają się coraz bardziej oswajane, a ja coraz peeniej i bardziej swojsko czuję się pisząc kod.

Technologie

HTML

HTML (Hypertext Markup Language) jest to kod używany do tworzenia struktury strony i jej zawartości. Na przykład treść może być uporządkowana w obrębie zestawu akapitów, listy punktowanych punktów lub tabel obrazów i danych.

CSS

Cascading Style Sheets, w skrócie CSS) – język służący do opisu formy prezentacji (wyświetlania) stron WWW. CSS został opracowany przez organizację W3C w 1996 r. jako potomek języka DSSSL przeznaczony do używania w połączeniu z SGML-em.

GIT

Git – rozproszony system kontroli wersji. Stworzył go Linus Torvalds jako narzędzie wspomagające rozwój jądra Linux. Git stanowi wolne oprogramowanie i został opublikowany na licencji GNU GPL w wersji 2.
